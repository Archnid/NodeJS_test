## NexaTech 正式上线部署指南

本指南以腾讯云轻量应用服务器为例，带你从零完成项目上线。

---

### 第一步：购买腾讯云轻量应用服务器

1. 打开 [腾讯云轻量应用服务器](https://cloud.tencent.com/product/lighthouse) 页面
2. 选择 **地域**：离你最近的 city（如北京、上海、广州）
3. 选择 **镜像**：系统镜像 → **Ubuntu 22.04 LTS**
4. 选择 **套餐**：最低配（2核2G）即可满足需求，约 50-70 元/月
5. 购买时长按需选择，完成付款

购买完成后，在控制台找到你的服务器，记下 **公网 IP 地址**。

---

### 第二步：设置防火墙规则

在腾讯云控制台 → 你的服务器 → **防火墙** 标签页，添加以下规则：

| 协议 | 端口 | 来源 | 说明 |
|------|------|------|------|
| TCP | 22 | 全部 | SSH 远程连接 |
| TCP | 80 | 全部 | HTTP 网页访问 |
| TCP | 443 | 全部 | HTTPS 加密访问 |

---

### 第三步：连接服务器

**方式一：腾讯云 WebShell（推荐新手）**

控制台 → 你的服务器 → 点击 **登录** 按钮，直接在浏览器中操作。

**方式二：本地终端 SSH**

```bash
ssh root@你的公网IP
```

输入购买时设置的密码（如忘记密码可在控制台重置）。

---

### 第四步：初始化服务器环境

将 `setup.sh` 上传到服务器并执行：

```bash
# 在本地终端执行（将文件传到服务器）
scp setup.sh deploy.sh root@你的公网IP:/root/

# SSH 登录服务器后执行
sudo bash /root/setup.sh
```

这个脚本会自动安装 Node.js 20、PM2、Nginx，并配置好反向代理。

---

### 第五步：部署项目

将项目代码上传到服务器：

```bash
# 方式一：scp 命令上传（在本地 NodeJS 目录执行）
scp -r app.js package.json routes views public root@你的公网IP:/root/nexatech/

# 方式二：如果服务器装了 Git，可以 clone 仓库
# git clone 你的仓库地址 /var/www/nexatech
```

然后在服务器上执行部署脚本：

```bash
sudo bash /root/deploy.sh /root/nexatech
```

或者直接手动部署：

```bash
cd /var/www/nexatech
npm install --production
pm2 start app.js --name nexatech
pm2 save
pm2 startup
```

---

### 第六步：验证

打开浏览器访问 `http://你的公网IP`，应该能看到 NexaTech 主页了。

常用运维命令：

```bash
pm2 status              # 查看应用状态
pm2 logs nexatech       # 查看实时日志
pm2 restart nexatech    # 重启应用
pm2 stop nexatech       # 停止应用
```

---

### 第七步（可选）：绑定域名

当你注册了域名并完成 ICP 备案后：

1. 在域名解析商处添加 A 记录，指向你的服务器 IP
2. 修改 Nginx 配置中的 `server_name _` 为你的域名
3. 申请免费 SSL 证书（Let's Encrypt）：

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d 你的域名.com
```

Certbot 会自动修改 Nginx 配置并设置 HTTPS，证书到期前会自动续期。

---

### 常见问题

**Q: 访问显示 502 Bad Gateway？**
检查 Node 应用是否在运行：`pm2 status`，如果没启动就 `pm2 restart nexatech`。

**Q: 访问超时打不开？**
检查腾讯云控制台的防火墙是否放行了 80 端口，以及服务器内部 ufw 是否放行。

**Q: 如何更新代码？**
将新代码上传后执行 `sudo bash /root/deploy.sh /root/nexatech` 即可一键更新。
